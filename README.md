# Netsells JIRA Badges for Pull Requests

> A GitHub App built with [Probot](https://github.com/probot/probot) that adds badges to Pull Requests to show the status of the branch in JIRA.

## Setup

```sh
# Install dependencies
yarn

# Run the bot
yarn start
```
